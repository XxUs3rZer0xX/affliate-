# Us3rZer0 Casino Affiliate Site

Casino affiliate blog targeting US sweepstakes casino players.
Built with Next.js 14 + Tailwind CSS. Deployed to us3rzer0.tech.

## Setup

```bash
npm install
npm run dev       # local dev at localhost:3000
npm run build     # production build
```

## Adding Your Affiliate Links

Edit `/data/casinos.js` and replace `YOUR_REF_CODE` with your actual affiliate IDs from each casino program.

## Pages

- `/` — Homepage with top casino rankings
- `/casinos` — Full casino listings
- `/bonuses` — No deposit bonus codes
- `/sweepstakes` — How sweepstakes casinos work (SEO page)
- `/blog` — Blog index
- `/disclaimer` — Legal disclaimer (required)
